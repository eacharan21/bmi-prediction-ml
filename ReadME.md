# BMI Prediction using Machine Learning

## 📌 Overview
This project predicts Body Mass Index (BMI) using machine learning techniques based on health and nutritional data.

## 🚀 Features
- Data preprocessing & EDA
- Regression model training
- Model evaluation

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## 📊 Results
Achieved accurate BMI prediction using regression models.
